SmartFraud Classifier

 Overview

SmartFraud Classifier is a machine learning project that detects fraudulent credit card transactions and identifies the most effective model among Decision Tree, Random Forest, and XGBoost.

 Key Highlights

* Built using Python and Scikit-learn
* Trained on real-world Kaggle fraud dataset (highly imbalanced)
* Compared multiple models to find the best performer
* Evaluated using Accuracy, Precision, Recall, and F1-score


 Models Used

* Decision Tree
* Random Forest
* XGBoost
 XGBoost showed the best performance in detecting fraud.


Tech Stack

Python | Pandas | Scikit-learn | XGBoost | Matplotlib

 
 Outcome
 
Developed a reliable fraud detection system and identified the most effective model for handling imbalanced transaction data.
